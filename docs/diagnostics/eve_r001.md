# EVE_R001: Agent request failed

The agent could not complete a request. The cause may be temporary provider availability, Gateway authentication, budget limits, or a tool failure.

## Fix

1. Retry once.
2. Find `EVE_R001` in the browser console or Vercel project logs.
3. Use the request ID and approximate failure time to correlate the event.
4. Do not include passwords, tokens, prompts, or private model output in a support request.

The structured log includes the diagnostic reason, fix, and documentation URL.
