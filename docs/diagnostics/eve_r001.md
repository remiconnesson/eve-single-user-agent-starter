# EVE_R001: Agent request failed

The agent could not complete a request. The cause may be temporary provider availability, Gateway authentication, budget limits, or a tool failure.

## Fix

1. Retry once.
2. Open **Diagnostics** in the app.
3. Click **Copy Report** and include it in the support request.
4. Include the approximate time of the failure, but do not include passwords, tokens, prompts, or private model output.

The support report includes deployment and configuration metadata without secret values.
